# TodoList App

Tässä ohessa TodoList-sovellus

## Käynnisty

1. Riippuvuudet:
   ```bash
   npm install
   ```

2. Kehityspalvelin:
   ```bash
   npm run dev
   ```

3. Avaa selain ja siirry osoitteeseen `http://localhost:5173`.

## Komponentit

- **App**: Pääkomponentti.
- **TodoList**: Sisältää syötekentät ja tilat.
- **TodoTable**: Renderöi todo-taulukon.
